### Setup

- Step 1: Pull down the repository
~~~
git clone https://github.com/anujoseph-popy/anu-youi-qa-test.git
~~~
- Step 2: CD into the directory
~~~ 
cd anu-youi-qa-test
~~~
- Step 3: Install npm packages
~~~ 
npm install 
~~~
- Step 4: Run cypress
~~~
npx cypress open
~~~
